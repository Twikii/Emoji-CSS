Emoji CSS

Vous voulez inclure des emoji dans votre code HTML ? Incluez simplement la feuille de style (minifiée) ci-dessous, puis ajoutez <i> balises pour insérer des emoji. C'est tout!
✈ Cliquez sur le code emoji et il sera copié dans votre presse-papiers.

{% highlight html %} {% assign emoji=site.data.emoji[104] %}{% include emoji-icon.html %} {% endhighlight %}

Par défaut, les emoji seront servis en PNG. Pour utiliser SVG à la place, utilisez le em-svg class au lieu de em. Cela s'adaptera mieux, mais la taille du fichier sera plus grande.
